## О проекте
Маленькая программа на C для демонстрации сортировок: пузырьковая, выбором, Шелла и слиянием.

## Сборка
Требуется компилятор C (gcc/clang). Пример:
```
gcc sort.c -o sort.exe
```

## Запуск
```
./sort.c
```
или
```
./sort.c numbers.txt
```
Программа создает случайный массив или использует данные из текстового файла, название которого дано аргументом, выводит отсортированный результат в stdout и txt файлы.
