# SortC

Простой и минимальный README для проекта на C, реализующего сортировку массива.

## О проекте
Маленькая программа на C для демонстрации одного или нескольких алгоритмов сортировки.

## Сборка
Требуется компилятор C (gcc/clang). Пример:
```
gcc sort.c -o sort.exe
```
Если несколько файлов:
```

## Запуск
```
./sort.c
```
или 
./sort.c numbers.txt
```

Программа создает случайный массив или использует массив данный на вход, выводит отсортированный результат в stdout и txt файлы.
